
# Portfolio website for Thiago

Portfolio responsivo em React, Vite, Tailwind CSS e Motion, pronto para deploy na Vercel.

## Rodando localmente

```bash
pnpm install
pnpm run dev
```

## Build de produção

```bash
pnpm run build
```

## Deploy na Vercel

Ao importar o repositório na Vercel, use as configurações padrão para Vite:

- Framework Preset: `Vite`
- Build Command: `pnpm run build`
- Output Directory: `dist`
